# Alena Cursor

Небольшая романтичная страница на React, TypeScript, Vite и Tailwind CSS. Сейчас реализован первый экран с мягким визуальным фоном и нижней кнопкой-заготовкой под будущий переход.

Основной целевой просмотр: Safari на iPhone 16 и iPhone 16 Pro Max.

## Команды

```bash
npm install
npm run dev
npm run build
npm run lint
```

Локальная разработка запускается на `http://localhost:5174`.

## Структура

- `src/App.tsx` - основной экран и нижняя кнопка под будущий переход.
- `src/index.css` - Tailwind, базовые стили и анимации.
- `public/favicon.svg` - иконка сайта.

## Проверка перед публикацией

```bash
npm run build
npm run lint
```
